# Yazdırma İçin Gerekli Görseller

Lütfen aşağıdaki görselleri `Defterdar/public/` klasörüne ekleyin:

## 1. turkbayrak.png
- **Konum:** Sol üst
- **Boyut:** 80px yükseklik
- **Format:** PNG
- **İçerik:** Türk bayrağı

## 2. yazi.png
- **Konum:** Orta üst (merkez)
- **Boyut:** 60px yükseklik
- **Format:** PNG
- **İçerik:** İÇDER logosu (yeşil logo)

## 3. tanzanya.svg
- **Konum:** Sağ üst
- **Boyut:** 80px yükseklik
- **Format:** SVG veya PNG
- **İçerik:** Tanzanya bayrağı

## Fallback
Görseller bulunamazsa otomatik olarak mevcut `icder.png` ve `cad.png` gösterilecek.

## Test
Görselleri ekledikten sonra:
1. Bir kurban oluştur
2. "Yazdır" butonuna tıkla
3. Görsellerin doğru göründüğünü kontrol et
